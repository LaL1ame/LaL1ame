# 软工作业自我介绍

## 摘要

就大学开始介绍吧，再往前显得冗长而拖沓。

先说结论，作者是一个缺乏足够行动力的理想主义者，很多事情上会幻想自己没有一点纰漏，但是真到了要做的时候总是提不起兴趣。

虽然偶尔会打一下鸡血，但是如果没有后续的推动，缺乏后劲，很快就会回到温暖的被窝开始躺尸。这种性格在很多事上都害了作者，各种半途而废，如今也有在尝试做出改变。
